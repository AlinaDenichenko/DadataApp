Для запуска программы запустите файл Dadata.exe

Приложение помогает пользователям найти точные координаты введенного адреса.

При первом запуске программы необходимо выбрать язык, на котором Вы будете работать, приложение поддерживает русский и английский языки (введите en или ru), язык по-умолчанию - русский. 

Далее Вам нужно ввести Ваш API ключ. 

Где получить API ключ: 

Для получения API ключа Вам необходимо зарегистрироваться в сервисе https://dadata.ru/api/suggest/address/, после чего API ключ будет доступен в Вашем личном кабинете по ссылке: https://dadata.ru/profile/#info

Для закрытия приложения введите 0

Основной функционал:

1. Для получения координат введите нужный адрес, приложение покажет возможные адреса, которые подходят по запросу.

2. Далее выберите циферное обозначение нужного адреса. Тогда приложение покажет точные координаты данного адреса.
